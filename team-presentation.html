Final Project: Tandem

Команда: GrowLab

Учебный командный проект RS School

Веб-приложение для подготовки к техническим собеседованиям с интерактивными виджетами и поддержкой AI ментора.

📌 Стек технологий

  • - Frontend: React, TypeScript, Zustand, CSS Modules, AI API
  • - Backend: REST API, база данных Mongo Atlas, Node.js/Express
  • - Dev Tools: Vite, ESLint, Biome, Git/GitHub

🏆 Командные достижения

  • Интерактивный фронтенд с виджетами, Dashboard и прогрессом
  • Разделение UI, бизнес-логики и глобального состояния
  • Продуманная архитектура движка виджетов и гибкая системой стратегий для различных типов виджетов
  • Backend с четким разделением слоев (БД, сервисы, HTTP-запросы, обработка ошибок)
  • Интеграция AI-ментора
  • Поддерживаемый и масштабируемый код

🎯 Проект включает:

  • - Практические задания по темам JavaScript, TypeScript, React и Node.js
  • - Разные форматы виджетов
  • - Поддержка AI-ментора для интервью
  • - Система прогресса пользователя

⭐️ Feature Components

  • Widget Engine — управление виджетами
  • Widget Strategies — стратегия работы виджетов
  • Quiz и Memory Match Game — интерактивные виджеты
  • AI Mentor — диалоговое интервью с подсказками
  • Login/Register — вход и создание аккаунта
  • Dashboard — прогресс и данные пользователя

📝 Принципы командной работы

  • Работа спринтами (1 неделя)
  • Работа через Pull Requests
  • Обязательный code review от ментора и команды
  • Merge только после аппрува
  • Регулярные созвоны (1 раз в неделю) и дополнительные созвоны с ментором по запросу
  • Архитектурные решения и дизайн обсуждались совместно
  • Фокус на читаемом и поддерживаемом коде
  • Единые правила кодстайла
  • Разделение задач и зон ответственности

Сложности и решения

  • Первый опыт работы с React
  • Разный уровень подготовки в команде
  • Изменения в составе команды
  • Длительные code review

Что это дало

  • Навык работы с чужим кодом
