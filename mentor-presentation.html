Учебный командный проект RS School

Веб-приложение для подготовки к техническим собеседованиям с интерактивными виджетами и поддержкой AI ментора.

🎯 Проект включает:

  • Практические задания по темам JavaScript, TypeScript, React и Node.js
  • Разные форматы виджетов: Quiz Widget и Match Memory Game
  • Поддержка AI-ментора для интервью: Groq API
  • Система прогресса пользователя

📌 Стек технологий

React TypeScript Zustand React Router REST API Zod CSS Modules Vite ESLint Git/GitHub Vitest Husky + lint-staged Biome classnames

📝 Моя работа

  • Разработка универсального Widget Engine с архитектурным паттерном Strategy
  • Фронтенд с виджетами Quiz Widget и Match Memory Game, Dashboard и прогрессом
  • Логика начисления очков, расчёт прогресса и открытие достижений
  • Интеграция с бэкендом через API-слой, создание эндпоинта /data
  • Деплой фронта через Vercel
  • Своими компонентами реализован лоадер, Toast уведомления, страница 500
  • Разработаны переиспользуемые UI-компоненты: кнопки, заголовки, карточки виджетов, иконки и т.д.

⭐️ Feature Components

  • Widget Engine — управление виджетами
  • Widget Strategies — стратегии работы виджетов
  • Quiz Widget — интерактивный квиз-виджет
  • Memory Match Game — интерактивная игра
  • Dashboard — прогресс и данные пользователя

🧭 Разделение ответственности для всех виджетов:

  • UI – слой представления: обработка пользовательских действий и отображение состояния

📦 Zustand Store

  • useUserStore — хранит данные пользователя
  • useScoreStore — управляет прогрессом, очками и достижениями
  • useSolvedWidgetsStore — отвечает за отслеживание уже решённых виджетов по типам
  • useToastStore — управляет отображением уведомлений пользователю
